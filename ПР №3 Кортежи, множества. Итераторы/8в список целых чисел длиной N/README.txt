Вариант 8
Дан список целых чисел длиной N, заполненный случайным образом.
Напишите программу, удаляющую повторяющиеся элементы при этом
сохраняя исходный порядок последовательности. Вывести количество
удаленных элементов для каждого повторяющегося числа.
Предусмотреть:
• ввод границ и размеров для набора
• вывод исходного набора на экран
• вывод результата
